# **History Taking App**

A simple, dynamic modern app to streamline history taking for doctors and medical students.



### **Features**

* 9 step wizard for structured history taking
* Real time autosave
* Download histories as PDF
* Light/ Dark mode toggle
* Works on both mobile and desktop



### **Tech Stack**

* React
* Vite
* Tailwind CSS 4
* jsPDF
* html2canvas



### **Getting Started**

1. npm install
2. npm run dev
